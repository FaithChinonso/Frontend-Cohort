# Tasty Bites — Learn Web Development from Scratch

Welcome! This course teaches you how to build a real website from absolute zero. By the end you will have built a **recipe blog called "Tasty Bites"** and you will understand HTML, CSS, JavaScript, TypeScript, and Tailwind CSS.

You don't need to know anything before starting. If you can use a mouse and type, you can do this.

---

## What you need before starting

1. **A computer** (Windows, Mac, or Linux — all work)
2. **A code editor** — install [Visual Studio Code](https://code.visualstudio.com/) (it's free)
3. **A web browser** — Chrome, Edge, or Firefox

That's it for now. We'll install more tools later when we get to TypeScript and Tailwind.

---

## How the course is structured

The course is split into **8 sections**. Each section is a folder that contains:

- `README.md` — the lesson. Read this first.
- `starter/` — files with the skeleton already set up. **You write your code here.**
- `solution/` — the finished version. **Only look at this after you try yourself.**

### The sections

| # | Section | What you learn |
|---|---------|---------------|
| 01 | [HTML Basics](01-html-basics/) | What HTML is. Tags, headings, paragraphs, links, images. |
| 02 | [HTML Semantic + Forms](02-html-semantic-forms/) | Lists, semantic tags, building a subscribe form. |
| 03 | [CSS Basics](03-css-basics/) | Colors, fonts, spacing, the box model. |
| 04 | [CSS Layout](04-css-layout/) | Flexbox, grid, and responsive design (mobile vs desktop). |
| 05 | [JavaScript Basics](05-js-basics/) | Variables, functions, conditions, arrays, loops. |
| 06 | [JS + The DOM](06-js-dom/) | Making the page react to clicks. Filter and search recipes. |
| 07 | [TypeScript](07-typescript/) | Adding types so your code is safer. |
| 08 | [Tailwind CSS](08-tailwind/) | Styling faster with utility classes. |

---

## The project: Tasty Bites

Throughout the course we build the same website piece by piece:

- **Sections 1–2** — build a single recipe page and a subscribe form (HTML only)
- **Sections 3–4** — make it look beautiful with CSS, build a homepage grid of recipe cards
- **Sections 5–6** — add a search bar and category filter that actually works
- **Section 7** — convert our JavaScript to TypeScript
- **Section 8** — rebuild the styles using Tailwind CSS

Each section builds on the one before. **Don't skip ahead.**

---

## How to work through each section

1. Open the section's folder in VS Code.
2. Read `README.md` from top to bottom.
3. Open the `starter/` folder. Open `index.html` in your browser by double-clicking it.
4. Follow the **"Your turn"** exercises in the lesson — edit the starter files.
5. Save the file, then refresh the browser to see your changes.
6. Stuck? Peek at `solution/` to compare.

---

## A few rules

- **Type the code yourself.** Don't copy-paste. Your fingers need to learn the shapes.
- **Break things on purpose.** Delete a tag, see what happens. Curiosity is how you learn.
- **Ask questions.** If something doesn't make sense, stop and ask.
